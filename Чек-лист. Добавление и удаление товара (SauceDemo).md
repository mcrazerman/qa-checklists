# 1. Добавление товара (кнопка «Add to cart»)
## Позитивные
- [x] Клик мышью по кнопке -> Кнопка заменяется на «Remove».
- [x] Клик мышью по кнопке -> На иконке корзины появляется красный бейдж со счетчиком (или цифра увеличивается на 1).
## Негативные и системные проверки:
- [x] Быстрое многократное нажатие на кнопку (даблклик) -> Добавляется только 1 экземпляр товара.
- [x] Обновление страницы (F5) после добавления -> Кнопка остается в состоянии «Remove», счетчик корзины не сбрасывается.
# 2. Удаление товара (кнопка «Remove»)
## Позитивные
- [x] Клик мышью по кнопке -> Кнопка заменяется на «Add to cart».
- [x] Клик мышью по кнопке (когда в корзине >1 товара) -> Счетчик на корзине уменьшается на 1.
- [x] Удаление последнего/единственного товара -> Красный бейдж со счетчиком полностью исчезает (граничные значения).
# 3. Проверки UI и счетчика корзины (Глобально)
- [x] Добавление 6 разных товаров по очереди -> Счетчик показывает точное число «6» (граничные значения).
- [x] Изменение окна браузера до размеров мобильного телефона -> Кнопки добавления/удаления не выходят за пределы экрана и не перекрывают цену.
