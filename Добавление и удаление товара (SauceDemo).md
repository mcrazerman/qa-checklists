# Добавление товара (кнопка «Add to cart»)
|ID|Проверка|Ожидаемый результат|Статус|Баг/ Комментарий|
|---|---|---|---|---|
|CL-1|Клик мышью по кнопке|Кнопка заменяется на «Remove».|✅||
|CL-2|Клик мышью по кнопке|На иконке корзины появляется красный бейдж со счетчиком (или цифра увеличивается на 1).|✅||
|CL-3|Быстрое многократное нажатие на кнопку (даблклик)|Добавляется только 1 экземпляр товара.|✅||
|CL-4|Обновление страницы (F5) после добавления|Кнопка остается в состоянии «Remove», счетчик корзины не сбрасывается.|✅||

# Удаление товара (кнопка «Remove»)
|ID|Проверка|Ожидаемый результат|Статус|Баг/ Комментарий|
|---|---|---|---|---|
|CL-5|Клик мышью по кнопке|Кнопка заменяется на «Add to cart».|✅||
|CL-6|Клик мышью по кнопке (когда в корзине >1 товара)|Счетчик на корзине уменьшается на 1.|✅||
|CL-7|Удаление последнего/единственного товара|Красный бейдж со счетчиком полностью исчезает (граничные значения).|✅||
# Сортировка товаров
|ID|Проверка|Ожидаемый результат|Статус|Баг/ Комментарий|
|---|---|---|---|---|
|CL-8|Открытие страницы каталога (состояние по умолчанию)|В дропдауне выбрано Name (A to Z). Карточки отсортированы по алфавиту от А до Я.|✅||
|CL-9| Выбор Name (Z to A)|Карточки перестраиваются по алфавиту в обратном порядке.|✅||
|CL-10|Выбор Price (low to high)|Цена каждой следующей карточки больше или равна предыдущей|✅||
|CL-11|Выбор Price (high to low)|Цена каждой следующей карточки меньше или равна предыдущей|✅||
|CL-12|Сортировка по цене при наличии товаров с одинаковой стоимостью|Товары с идентичной ценой сортируются по алфавиту.|✅||
|CL-13|Обновление страницы после применения любой недефолтной сортировки|Выбранный тип сортировки сохраняется, карточки остаются в заданном порядке|✅||
