# 1. Кнопка Continue
## Позитивные
- [x] Все три поля (`First Name`, `Last Name`, `Zip/Postal Code`) заполнены -> клик Continue -> Успешный переход на страницу «Checkout: Overview».
## Негативные
- [x] Оставить `First Name` (остальные заполнены) -> клик Continue -> Ошибка `Error: First Name is required`.
- [x] Оставить `Last Name` (остальные заполнены) -> клик Continue -> Ошибка `Error: Last Name is required`.
- [x] Оставить `Zip/Postal Code` (остальные заполнены) -> клик Continue -> Ошибка `Error: Zip/Postal Code is required`.
- [x] Оставить все поля пустыми -> клик Continue -> Ошибка `Error: First Name is required`.

# 2. Кнопка Cancel
## Позитивные
- [x] Клик по кнопке «Cancel» -> Возврат на страницу корзины («Your Cart»).
## Негативные
- [x] Ввод любых данных в поля -> Клик «Cancel» -> Повторный переход со страницы корзины в Checkout -> Поля формы отображаются пустыми (данные не сохранились).

# 3. UI
- [x] Фокус: Клик внутри каждого поля ввода -> Поле выделяется визуально.
- [x] Граничные значения: Ввод очень длинной строки в поля -> Текст скроллится, вертска не искажается
- [x] Адаптивность: Сужение окна до размеров смартфона -> Поля ввода и кнопки корректно масштабируются и не выходят за пределы экрана.
